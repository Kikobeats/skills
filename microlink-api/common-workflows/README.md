# Microlink Common Workflows

Use these workflow files as focused, copy-paste-ready recipes.

- [metadata.md](metadata.md): extract normalized metadata from a URL
- [screenshot.md](screenshot.md): generate a page screenshot
- [pdf.md](pdf.md): generate a PDF
- [embed.md](embed.md): return assets directly with `embed`
- [scraping.md](scraping.md): extract custom fields with `data` rules
- [all-links.md](all-links.md): extract all links from a target URL
- [all-images.md](all-images.md): extract all images from a target URL
- [page-markdown.md](page-markdown.md): turn a whole page into markdown
- [page-text.md](page-text.md): turn a whole page into text
- [page-html.md](page-html.md): turn a whole page into HTML
